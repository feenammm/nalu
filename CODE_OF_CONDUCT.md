# 行为准则

## 我们的承诺

为了营造一个开放和友好的社区环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、种族、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、宗教信仰或性取向如何，参与本项目都免受骚扰。

## 我们的准则

有助于营造积极环境的行为包括：

- 对他人保持友善与包容
- 尊重不同观点与经历
- 虚心接受建设性意见
- 为自己的错误承担责任并道歉
- 关注对整个社区最有利的事情

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱或贬损性评论，以及人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在社区中可合理视为不当的行为

## 执行

如发现不当行为，可向项目维护者举报。所有投诉将得到及时、公正的审查和处理。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.0 版本。
