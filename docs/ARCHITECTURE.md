## 详细清单

### Client 端（macOS 桌面 / Android / iOS）

运行在用户设备上，**离线也能完整使用**（同步功能除外）。

| 模块 | 技术 | 说明 |
|------|------|------|
| 应用壳 | Tauri 2.11.2 | 进程管理、窗口管理、系统托盘 |
| UI 渲染 | Vue 3 + Vue Router + 系统 WebView | 所有界面（命令面板、设置、各功能页） |
| 状态 | Pinia | 全局警报、剪贴板监听、语言和 AI 配置 |
| 样式 | Tailwind CSS 4 | 组件样式 |
| Markdown 编辑 | Milkdown 7.21 | 笔记/任务的富文本编辑 |
| 本地数据库 | SQLite (rusqlite 0.40) | 任务、笔记、日程、配置等所有结构化数据 |
| MySQL 客户端 | mysql_async 0.37 | 连接用户的本地 MySQL 做导入导出 |
| 剪贴板 | tauri-plugin-clipboard-manager | 系统剪贴板读写 |
| 全局热键 | tauri-plugin-global-shortcut | Cmd+Space 唤起命令面板 |
| 系统通知 | tauri-plugin-notification | 番茄钟提醒、闹钟、日程提醒 |
| 文件操作 | tauri-plugin-fs + dialog | Markdown 文件读写、目录选择 |
| 插件系统 | Rust 自研 | 插件加载、命令注册、插件间通信 |
| CRDT 客户端 | automerge 0.10 | 生成/应用变更集，本地数据副本 |
| AI 调用 | reqwest (HTTP) | 调用本地 Ollama 或云端 API |
| 定时器/闹钟 | tokio + tauri-plugin-notification | 番茄钟倒计时、日程定时触发 |

### Server 端（Linux VPS / 自建服务器）

只部署两个服务，**都极其轻量**。

| 模块 | 技术 | 说明 |
|------|------|------|
| 同步中继 | axum 0.8.9 + WebSocket | 中转 CRDT 变更集，广播给在线设备 |
| 变更集存储 | SQLite 或文件系统 | 暂存离线期间的变更，设备上线后推送并清理 |
| Ollama（可选） | Ollama + 开源模型 | AI 推理服务，仅当你 VPS 有足够算力时部署 |

### 不在 Server 端运行的东西

| 东西 | 为什么不在 Server |
|------|------------------|
| SQLite 主数据库 | 本地优先，数据在设备上 |
| 业务逻辑 | 全在客户端 Rust 后端 |
| 文件存储（.md） | 本地文件系统 |
| MySQL 操作 | 直连用户本地 MySQL |
| UI 渲染 | WebView 在客户端 |
| 插件运行时 | 客户端进程内或子进程 |

## 数据流向

```
用户在 macOS 上添加一个待办
        │
        ▼
   Vue UI ──IPC──► Rust 后端
        │                │
        │                ├──► SQLite 写入（本地持久化）
        │                │
        │                └──► Automerge 生成变更集
        │                         │
        │                         ▼
        │              WebSocket 推送到 Relay Server
        │                         │
        │                         ▼
        │              Relay 广播给其他在线设备
        │                         │
        │                         ▼
        │              Android/iOS 客户端收到变更集
        │                         │
        │                         ▼
        │              本地 Automerge 应用变更 → SQLite 更新
        │                         │
        │                         ▼
        └──────────────────── 移动端 UI 刷新
```

## Server 部署规格参考

同步中继服务非常轻，最便宜的 VPS 就能跑：

| 指标 | 要求 |
|------|------|
| CPU | 1 核足够 |
| 内存 | 128 MB 足够 |
| 磁盘 | 1 GB（变更集很小，定期清理） |
| 带宽 | 极低（只传 CRDT 变更二进制，不传业务数据） |
| 系统 | Ubuntu 22.04+ / Debian 12+ |
| 端口 | 443 (HTTPS/WSS) |

Ollama 如果部署在 Server 上，则需要 GPU 或至少 16GB 内存的 CPU 实例。
